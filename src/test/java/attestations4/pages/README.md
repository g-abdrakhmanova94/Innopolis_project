Промежуточная аттестация

Задание:
1.	Напишите автотесты на сервис https://www.saucedemo.com/.
2.	Сценарии:
2.1.Успешная авторизация (standard_user)
2.2.Авторизация заблокированного пользователя (locked_out_user)
2.3.e2e-сценарий (под пользователями standard_user и performance_glitch_user):
 2.3.1.Авторизоваться
 2.3.2.Добавить в корзину товары:
-	Sauce Labs Backpack
-	Sauce Labs Bolt T-Shirt
-	Sauce Labs Onesie
  2.3.3. Перейти в корзину
  2.3.4. Проверить отображение 3 товаров 
  2.3.5. Нажать Checkout
  2.3.6. Заполнить форму валидными данными
  2.3.7. Нажать Continue
  2.3.8. Проверить, что сумма заказа = $58.29
  2.3.9. Нажать Finish

Требования:
1.	Работа принимается в виде github-репозитория
2.	Используется Selenide или Selenium
3.	Применен подход PageObject
4.	Формируется Allure-отчет

Критерии оценивания:
1.	Код компилируется и запускается
2.	Запускается браузер и выполняются автотесты
3.	Автотесты проходят в зеленом статусе (не падают)
4.	Формируется понятный читаемый Allure-отчет
5.	В репозитории нет лишних файлов/папок
